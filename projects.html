Projects

Malaria Detection Using Deep Learning

This project involves detecting malaria cells with high accuracy using deep learning and image processing techniques. We aimed to identify infected cells in microscopic images to contribute to early diagnosis and improve healthcare. Significant advancements in disease detection were achieved by leveraging deep learning models and image processing techniques.

Technologies Used: Python, TensorFlow, OpenCV

Sentiment Analysis of Restaurant Reviews

In this project, we analyzed positive and negative reviews about restaurants using Natural Language Processing (NLP) techniques. The goal was to assess customer sentiment and evaluate the strengths and weaknesses of restaurants. Through sentiment analysis, valuable insights into restaurant performance and customer satisfaction were obtained.

Technologies Used: Python, NLTK, Scikit-Learn
